# Misc Background Images

This folder contains background images that will be automatically used throughout the website.

## How It Works

Any image placed in this folder will be used as a background image in various sections of the website:

- **Hero Section**: The main hero/landing section
- **Section Backgrounds**: Trust section, services section, gallery section
- **Body Background**: Overall page background (if multiple images, they'll rotate)

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Image Recommendations

- **Size**: 1920px wide or larger for best quality
- **Aspect Ratio**: 16:9 or wider works well
- **File Size**: Optimize images for web (under 500KB recommended)
- **Style**: Wood textures, workshop scenes, or abstract wood patterns work great

## Usage

Simply drop your background images into this folder. The website will automatically detect and use them.
